## Фаза 1. Главная страница `/`

> Конверсионная страница №1. Задача: страх → доверие → заявка.

- [ ] **1.1** Герой-блок
  - Заголовок H1 с акцентом на боль: штрафы, отказы, сложность
  - Подзаголовок: решение через нас — быстро, с гарантией
  - CTA «Оставить заявку» + «Рассчитать стоимость»
  - Фоновое фото/видео: грузовик в Москве
  - Цифры в одну строку: 10 лет / 15 000+ пропусков / 70% продлений

- [ ] **1.2** Блок «Зачем нужен пропуск»
  - Зоны ограничений (МКАД / ТТК / СК) — визуальная карта
  - Размеры штрафов за проезд без пропуска
  - Ссылки на страницы услуг по каждой зоне

- [ ] **1.3** Калькулятор стоимости
  - Выбор: зона (МКАД/ТТК/СК), тип (годовой/временный), кол-во машин
  - Мгновенный расчёт с учётом скидок за объём
  - CTA «Оформить» после расчёта

- [ ] **1.4** Блок «Как это работает» (этапы)
  - 4 шага: Заявка → Документы → Подача → Пропуск
  - Иконки + краткое описание каждого шага
  - Акцент: «Вы просто отдаёте документы — мы делаем всё остальное»

- [ ] **1.5** Блок «Почему мы»
  - 10 лет на рынке (с 2016)
  - Гарантия результата
  - Берём сложные случаи (отказы, повторные подачи)
  - Скорость — быстрее конкурентов

- [ ] **1.6** Блок отзывов
  - 3–4 карточки из API Яндекс.Карт / 2ГИС
  - Рейтинг (звёзды) + текст + имя
  - Ссылка «Все отзывы →»

- [ ] **1.7** Блок «Частые вопросы» (3–5 из FAQ)
  - Аккордеон с микроразметкой FAQPage
  - Ссылка «Все вопросы →»

- [ ] **1.8** Финальный CTA-блок
  - «Оставьте заявку — мы перезвоним за 15 минут»
  - Форма: имя, телефон, кнопка
  - Или: телефон + WhatsApp/Telegram

- [ ] **1.9** SEO-оптимизация главной
  - Title: «Пропуск в Москву для грузовиков — оформление за 1 день | Инфологистик-24»
  - Description: с ключами «пропуск МКАД», «грузовой пропуск Москва»
  - Schema.org: LocalBusiness + AggregateRating
